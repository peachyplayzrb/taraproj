## Summary
Database and data preparation workshop sheet for module 6CS030. Covers:
1. Analysing and cleaning data loaded into Oracle SQL (REGION and POPULATION tables from Workbook 1)
2. Removing duplicate columns via `ALTER TABLE DROP COLUMN`
3. Reshaping cross-tabulated population data (wide → long format) using Excel
4. Creating a `DataUpdate` worksheet with reshaped year/population data
5. Using Excel charting and the Oracle SQL Developer tool for data exploration

The dataset is a UK regional population time-series (`Resident Pay` spreadsheet). This is an Oracle/SQL/Excel data cleaning exercise for a different module assignment, unrelated to information retrieval.
